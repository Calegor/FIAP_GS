Sobre o projeto

Conheça nossa missão

Tecnologia sem Barreiras

A revolução da Inteligência Artificial não deve deixar ninguém para trás. Nossa missão é desmistificar o linguajar corporativo atual, traduzindo termos técnicos e complexos para uma linguagem acessível. Transformamos a ameaça da obsolescência na oportunidade de dominar novas ferramentas.

Valorizando o Humano

Máquinas processam dados, mas apenas humanos têm empatia e criatividade. A Plataforma ABRIR foca no desenvolvimento de habilidades e na inclusão de grupos vulneráveis, garantindo que a tecnologia sirva para reduzir desigualdades, e não para concentrar oportunidades nas mãos de quem já as possui.

Requalificação Estratégica

Não basta estudar; é preciso aprender o que o mercado pede. Utilizamos análise de dados para diagnosticar suas habilidades atuais e criar trilhas de aprendizado personalizadas, conectando você aos cursos e certificações que realmente abrem portas para o futuro.

Trabalho Digno e Renda

Nosso objetivo final é a geração de renda e trabalho decente. Conectamos talentos requalificados a vagas em alta e oferecemos orientação sobre direitos na Gig Economy, construindo uma ponte segura e ética entre você e sua nova carreira.

Tecnologias utilizadas

Front-end

  • HTML
  • CSS
  • JavaScript

Estruturação de conteúdo semântico com HTML, estilização responsiva e moderna com CSS e interatividade dinâmica com JavaScript puro, garantindo uma experiência de usuário fluida e acessível.

Back-end

  • Python
  • Java

Arquitetura híbrida e robusta. Utilizamos Java para a construção de APIs seguras e lógica de negócios, enquanto o Python alimenta nossos algoritmos de Machine Learning para o cálculo de risco profissional e recomendações personalizadas.

Database

  • Data Modeler

Modelagem relacional de alta precisão com Data Modeler. Estruturamos relacionamentos complexos entre Usuários, Habilidades (Skills) e Cursos para garantir integridade referencial e performance nas consultas de diagnóstico de carreira.

Integrações

  • WatsonX
  • Telegram

O poder da IA e de Chatbots com IBM WatsonX, criando um "Coach de Carreira" inteligente. A integração via Telegram garante acessibilidade total, permitindo interações por texto e voz para inclusão digital.

Serviços externos

  • Vercel

Hospedagem e deploy do front-end/aplicação no Vercel, aproveitando sua CDN global e otimizações automáticas para garantir alta performance e disponibilidade.

Design

  • Canva

Utilização do Canva para criação e edição de ativos visuais, como ícones e banners, garantindo um UI coerente, moderno e profissional para a aplicação.
